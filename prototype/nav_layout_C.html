方案 C 说明 — 侧边栏折叠 + 顶栏极简

左侧边栏：NiceCut logo → 项目（可切换）→ 全部/已收藏 → 视频源列表（Checkbox多选）→ 底部灵敏度

顶栏：项目名 + 镜头数 → 排序切换 → 景别切换

特点：

  • 侧边栏模式，视频源列表垂直排列，一目了然，适合多视频源
  • 侧边栏可折叠收起（点击 ◀），收起后只显示极简顶栏 + ☰ 按钮展开
  • 灵敏度放在侧边栏底部，不占顶栏空间
  • 顶栏极其简洁：只有标题 + 排序 + 景别
  • 侧边栏中「全部」和「已收藏」互不干扰
  • 视频源 Checkbox 列表可一次看到所有源和对应镜头数

适合场景：视频源多（10+）、屏幕较宽、需要频繁切换各种筛选组合

注意：这个方案改动最大，会引入侧边栏布局。如果不想大改现有布局，可以选 A 或 B。
